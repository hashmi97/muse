# Muse - Wedding Planning Platform

A romantic, minimal wedding planning platform designed for couples to plan their wedding together.

## Features

### 11 Complete Screens (All 1440px wide)

1. **Landing Page** - Elegant entrance with soft gradient background
2. **Signup & Partner Invite** - Clean signup flow with partner invitation modal
3. **Onboarding** - Event selection with 6 customizable wedding events
4. **Dashboard** - Main workspace with budget summary, upcoming events, and activity feed
5. **Calendar View** - Monthly calendar with event color coding
6. **Budget Planner** - Event-based budget tracking with category breakdowns
7. **Honeymoon Planner** - Complete honeymoon planning with flights, hotels, and activities
8. **Event Page** - Tabbed interface with Overview, Tasks, Budget, Mood Board, and Notes
9. **Mood Board** - Event-level inspiration gallery
10. **Global Gallery** - Masonry-style inspiration gallery with filters
11. **Activity Feed** - Chronological feed of all couple activities

## Design System

- **Colors**: Soft rose (#ffa3b8), peach (#ffd9c4), and grey tones
- **Typography**: Playfair Display (headings), System fonts (body)
- **Spacing**: 8px grid system
- **Corners**: 12px rounded for cards
- **Shadows**: Subtle, soft shadows throughout

## Navigation

Navigate between screens using the top navigation bar:
- Dashboard
- Calendar
- Budget
- Honeymoon
- Gallery
- Activity

## Getting Started

1. Visit the landing page
2. Click "Start Planning"
3. Create your workspace
4. Invite your partner
5. Select your wedding events
6. Start planning!
